Second Opinion

Mix Codex, Claude Code, OpenCode, and Grok Build as opt-in subagents. Each agent keeps its own CLI, account, model access, and safety settings.

Agent Matrix

Second Opinion installs native skills that teach each agent how to call another installed agent for a scoped task.

Codex

Implementation and tests

Useful for repo-wide edits, patch review, and OpenAI/Codex ecosystem work.

Claude Code

Architecture and review

Useful for design review, edge cases, refactors, and reasoning-heavy analysis.

OpenCode

Provider-flexible tasks

Useful when you want the user's configured OpenCode models and providers involved.

Grok Build

Fast exploration

Useful for broad investigation, implementation spikes, and xAI/Grok-specific tasks.
